My name is Markus and I'm a mechatronics student, currently finishing my 4th year at LTH (Lunds Tekniska Högskola). I have a passion for bridging the gap between software and hardware, and enjoy building functional and robust robotic prototypes. 

My GitHub serves as a portfolio for all my prior projects, including everything between service robotics and my liquorice mustard venture. 



- 🔭 I’m currently working on a VINNOVA project called AORTA - Advanced Offloading of Real-Time Applications, together with Ericsson, Cognibotics and Mälardalen University. 

The aim of the project is to explore different options to offload real-time control robotic applications to the cloud. My objective in this project is to design and implement a Robot Operating System (ROS)-based control algorithm for the ABB YuMi robot. 
The task is to build a program where YuMi autonomously probes, grabs and lifts a tray from a table, and transport it safely to a given location. This should be done without knocking over anything potentially placed at the tray, such as a wine glass.
